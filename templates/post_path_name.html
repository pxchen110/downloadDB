Extracting pathway gene list from Public databases

Available database: KEGG and Harmonize databases

Input example:

KEGG pathway url: https://www.kegg.jp/entry/hsa05130
Harmonize geneset page url: https://maayanlab.cloud/Harmonizome/gene_set/BRCA2/Pathway+Commons+Protein-Protein+Interactions
Harmonize api url: https://maayanlab.cloud/Harmonizome/api/1.0/gene_set/BRCA2/Pathway+Commons+Protein-Protein+Interactions

Input section:

User can choose one of the input format:

For only one or two queries, user can select the database source, type query pathway, and click submit

For list of queries, user can select the database source, upload text file with list of queries urls
